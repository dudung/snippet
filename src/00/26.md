# 0026
Shape function for line element.

$$\tag{1}
\varphi(x) = N_1(x) \ \varphi_1 + N_2(x) \ \varphi_2.
$$

$$\tag{2}
\varphi_i = \varphi(x_i), \ \ i = 1, 2, 3.
$$

$$\tag{3}
\varphi(x) = \sum_{i = 0}^m a_i x^i.
$$
