# 0025
Shape function for triangle element.

$$\tag{1}
\varphi(x,y) = N_1(x,y) \ \varphi_1 + N_2(x,y) \ \varphi_2 + N_3(x,y) \ \varphi_3.
$$

$$\tag{2}
\varphi_i = \varphi(x_i, y_i), \ \ i = 1, 2, 3.
$$

$$\tag{3}
\varphi(x, y) = \sum_{i = 0}^m a_i x^i \sum_{j = 0}^m b_j y^j
$$
