# 0033
Solve 3 uknowns with 3 equations for $N_3(x, y)$ from $N_2(x, y)$?

$$\tag{1}
b_3 = \left[ \frac{ (y_1 - y_2) }{ (x_3 - x_1)(y_1 - y_2) - (x_1 - x_2)(y_3 - y_1) } \right].
$$

$$\tag{2}
c_3 = - b_3 \left( \frac{x_1 - x_2}{y_1 - y_2} \right).
$$

$$\tag{3}
a_3 = 1 - b_3 x_3 + c_3 y_3.
$$
